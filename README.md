## Các phím di chuyển
Các phím di chuyển:

w: đi thẳng  
x: đi lùi  
a: sang trái  
d: sang phải  
q, e, z, c: đi chéo  
j: quay trái tại chỗ  
l: quay phải tại chỗ  
s hoặc space: dừng
